* ### Route: https://kanban-fix-asli.firebaseapp.com/tasks
    - Method : GET
    - Response : 
      * code 200 :
      ```
              [
                {
                  "id": 1,
                  "title": type data string,
                  "status": type data string,
                  "category": type data string,
                  "date": type data date,
      ```

      * code 500 :
        ```
          {
            "err": {
              "code": 500,

            },
            "msg": "internal server error"
          }
        ```
